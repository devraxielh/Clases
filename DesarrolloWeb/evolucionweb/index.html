La Evolución de la Web

Desde sus orígenes hasta la Web 3.0

¿Cómo ha cambiado la web?

La web ha pasado de ser un espacio estático de información a un entorno inteligente, interactivo y descentralizado. Explora su evolución cronológica a continuación:

🌐 Web 1.0 (1990 - 2000) ▼

La primera etapa de la web, también llamada web estática:

  • Solo lectura
  • Contenido fijo y páginas HTML simples
  • Interacción limitada del usuario
  • No existían redes sociales

💬 Web 2.0 (2000 - 2020) ▼

Etapa de interacción social y contenido generado por el usuario:

  • Lectura y escritura
  • Redes sociales y blogs
  • Aplicaciones web dinámicas
  • Centralización de plataformas (Google, Facebook)

🤖 Web 3.0 (2020 - Presente) ▼

La web semántica, inteligente y descentralizada:

  • Datos conectados y comprensión contextual
  • Inteligencia Artificial y blockchain
  • Control de datos por los usuarios
  • Metaverso y experiencias inmersivas
