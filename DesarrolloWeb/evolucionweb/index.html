OVA: Evolución de la Web

Web 1.0

Es la primera etapa de la web. Se caracterizaba por páginas estáticas, sin interacción con el usuario. Los contenidos eran únicamente informativos, como una enciclopedia digital.

Web 2.0

Introdujo la interacción del usuario con las páginas. Aparecen las redes sociales, blogs y plataformas colaborativas. El usuario ahora también crea contenido.

Web 3.0

También conocida como la web semántica. La información se organiza de forma más inteligente gracias al uso de inteligencia artificial y tecnologías como blockchain.

Web 4.0

Visión futura donde la web se vuelve ubicua, conectando todos los dispositivos inteligentes. Se caracteriza por experiencias inmersivas con realidad aumentada, IoT y asistentes virtuales.
