¿Qué es JSON?

Explora la estructura, utilidad y sintaxis de JSON con ejemplos prácticos e interacciones.

📌 Significado de JSON

JSON significa JavaScript Object Notation. Es un formato ligero de intercambio de datos. Fácil de leer y escribir para los humanos, y fácil de analizar y generar por las máquinas.

🔧 Estructura básica de JSON

Un archivo JSON contiene:

  • Pares clave-valor
  • Listas (arreglos)
  • Objetos anidados
{
  "nombre": "Ana",
  "edad": 25,
  "habilidades": ["HTML", "CSS", "JavaScript"],
  "activo": true
}

🧪 Prueba interactiva

Modifica los valores en el campo de abajo y haz clic en Validar para verificar si es un JSON válido.

🌍 ¿Dónde se usa JSON?

  • APIs RESTful
  • Almacenamiento de configuración
  • Intercambio de datos entre cliente y servidor
  • Persistencia en bases de datos NoSQL como MongoDB
